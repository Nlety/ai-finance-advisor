# AI 个人理财师

本项目由阿里云ESA提供加速、计算和保护

<img src="https://img.alicdn.com/imgextra/i3/O1CN01H1UU3i1Cti9lYtFrs_!!6000000000139-2-tps-7534-844.png" width="50%" />

💰 智能财务规划与消费决策助手

## 功能特点

- **月度预算规划** - 基于收入和支出，生成合理的预算分配方案
- **储蓄目标计划** - 为买房、买车等目标制定可行的储蓄计划
- **消费决策分析** - 分析大额购买的性价比和财务影响
- **财务健康诊断** - 诊断收支结构，给出改善建议

## 技术栈

- 纯原生 JavaScript（无框架）
- Tailwind CSS（CDN）
- OpenAI-compatible API（GLM-4-Flash）
- EdgeKV 云端存储
- CryptoJS 配置加密

## 使用方法

1. 选择规划类型（月度预算/储蓄目标/消费决策/财务诊断）
2. 填写相关财务信息
3. 点击"生成AI建议"获取专业理财建议
4. 保存方案到历史记录

## 特色设计

- **左右分栏布局** - 左侧输入，右侧展示AI建议
- **顶部工具栏** - 保存/历史/设置按钮集中在顶部
- **金绿配色** - 象征财富和成长的渐变色
- **表单式交互** - 结构化数据输入，精确的财务分析
- **免责声明** - 明确提示仅供参考，不构成专业投资建议

## AI 配置

Temperature: 0.4（精确计算和理性分析）

## 注意事项

本工具提供的财务建议仅供参考，不构成专业投资建议。重大财务决定建议咨询专业理财顾问。

## 开发

本应用遵循统一的三层架构：
- `index.html` - 前端页面
- `js/app.js` - 应用逻辑
- `js/aiService.js` - AI服务封装
- `js/storageService.js` - 存储服务
- `edge-functions/` - 边缘函数后端
